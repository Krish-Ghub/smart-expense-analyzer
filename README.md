# 🚀 Smart Expense Analyzer

A full-stack application that helps users track expenses and gain insights into their spending behavior.

---

## 📱 Features

- Expense tracking app built with Flutter  
- Persistent data storage using SQLite  
- Category-based expense visualization  
- In-app analytics and insights  
- Interactive dashboard using Streamlit  

---

## 🧠 Tech Stack

- Flutter (Dart)
- SQLite (sqflite)
- Python (Pandas, Matplotlib)
- Streamlit

---

## 📊 Insights Generated

- Category-wise spending analysis  
- Detection of overspending patterns  
- Daily spending trends  
- Smart recommendations
